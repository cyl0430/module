Grunt + Require + Sass + Watch 模板
  • 此模板执行的命令 :
  • step1 : 每次新建.sass 文件后,均需执行 grunt sass 来创建相对应的 .css 文件
  • step2 : 每次在 src 目录下新建完 .JS 文件后,均需执行 grunt require 在js目录下创建相对应的js文件
  • step3 : grunt watch 开启监听,即可开始编译
  • 备注 :
  • 1.如果你是从我的GitHub上下载下来的,需要先 cnpm install 下载依赖
  • 2.如果js代码报错,watch监听会自动终止运行,此时需要重新开启监听 grunt require + grunt watch
  • 目录介绍 :
  • XXX 项目名称
  • ---public 存放前端所用全部文件
  • ---static 存放前端所用全部静态资源
  • ---css 存放页面所用的压缩后的css文件
  • ---js 存放页面所用的压缩后的js文件
  • ---scss 存放前端编写的源文件 css样式
  • ---src 存放前端编写的源文件 js代码
  • ---require.js 引入 require.js ,实现js代码模块化
  • ---view 视图
  • ---index 存放页面分类 html文件
  • ---Gruntfile.js 存放grunt的配置,包括但不限于 scss/watch/代码压缩 等环境
  • ---package.json 存放本项目所引用的npm插件,也是npm所要执行的命令
